## Description

[Beholder multi-coin bot](https://beholder.julianboetto.lat) to create automations and analyze in real time any cryptocurrency on the Binance exchange.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

Only the first time (initial configuration)
$ npm run db:restart:test
$ npm run db:migrate:test

# test coverage
$ npm run test:cov
```
